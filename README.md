# ArtOfTesting Selenium Automation Framework

## Tech Stack
- Java
- Selenium WebDriver
- Page Object Model (POM)
- Maven
- TestNG (planned)

## Features
- Login automation
- Dynamic product selection
- Add to cart & checkout
- Property file configuration
- Reusable utilities

## How to Run
1. Clone repository
2. Update ArtOfTesting.properties
3. Run BuyingSingleItem.java
